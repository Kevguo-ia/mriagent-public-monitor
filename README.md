# MRIAgent Public Monitor

Public, read-only aggregate dashboard for the 3,906-study formal experiment.

- No case-level table or identifiers are published.
- The page shows center-level stages, intermediate artifact counts, LGE completion, report throughput and ETA, worker state, GPU snapshots, and unresolved error counts.
- The `monitor_safe_v2` privacy gate rejects paths, identifiers, clinical content, private logs, and credentials before publication.
